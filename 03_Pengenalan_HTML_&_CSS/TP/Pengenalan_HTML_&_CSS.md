HTML (Hypertext Markup Language) adalah bahasa markah (bukan bahasa pemrograman) untuk menyusun dan menampilkan dokumen Web. Dikembangkan oleh Tim Berners-Lee pada 1993, HTML adalah inti teknologi WWW bersama HTTP dan URI/URL. Dulu populer HTML 4.01 dan XHTML 1.0, kini digunakan HTML5.

Spesifikasi HTML sempat menjadi konflik antara W3C dan WHATWG. WHATWG dibentuk pada 2004 karena W3C fokus ke XHTML (revisi HTML4 berbasis XML). Pada 2019, W3C setuju menyerahkan pemeliharaan spesifikasi HTML sepenuhnya kepada WHATWG.
